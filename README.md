# ふしぎモンスター・チューニング大会

小1前後の子が、ステータスを変えながら遊べるブラウザゲームです。
QRコードのリンク先に置きやすいよう、依存なしの静的ファイルだけで作っています。

## ファイル

- `index.html`: 画面の骨組み
- `styles.css`: レイアウトと見た目
- `game.js`: ゲーム本体、AI対戦、ステータス、描画、音
- `assets/monsters/`: 背景抜きしたモンスター画像
- `assets/monsters/source/`: 生成直後の元画像

## 遊び方

### 自由に遊ぼう

好きな種目、タイプ、すがた、パラメータで自由に試せます。
種目ごとにステータスの効き方が違います。

### AIと対戦

最初に決めたタイプ、すがた、パラメータを固定して3種目を連続で戦います。
AIは相手ごとに得意な種目と苦手な種目があり、1つの最強チューニングが見つかりにくい作りです。
ページ更新やリセットのたびに、AIの型、能力、目標スコアが少し変わります。

## 種目

- おたからあつめ: 速さ、曲がりやすさ、体の大きさのバランスが大事です。
- ころころすもう: 力と重さが効きますが、重すぎると追いかけにくくなります。
- ぴょんジャンプ: ジャンプ力、速さ、重さ、空中での曲がりやすさが絡みます。

## 改造の軸

- ステータスは合計24ポイントまでです。
- タイプは三すくみです。
  - スイスイは、どっしりに強い
  - どっしりは、くるりに強い
  - くるりは、スイスイに強い
- 速さを上げすぎると曲がりにくくなります。
- 重さを上げると押し合いに強くなりますが、ジャンプや加速が弱くなります。
- 体を大きくするとアイテムを取りやすくなりますが、ぶつかりやすくなります。

## モンスターのすがた

- ツノドラゴン: かっこいい
- ライガー: かっこいい
- もこぷに: かわいい
- キラコロ: かわいい

ゲーム中は画像スプライトを優先して表示します。
画像が読み込めない環境では、従来のキャンバス描画モンスターに戻ります。
移動方向に応じて左右反転します。

## 画像と権利面

画像生成時のプロンプトでは、既存作品名、既存キャラクター名、ブランド名、作家名を使っていません。
ただし、公開前には有名キャラクターに似すぎていないか目視確認してください。
特に「電気」「小型モンスター」などの一般モチーフは既存作品を連想しやすいため、ライガーは色とシルエットをオリジナル寄りに調整しています。

## AIの作戦

- ビュンビュンAI: おたからが得意、すもうが苦手
- ドスコイAI: すもうが得意、ジャンプと回収が苦手
- ピョンピョンAI: ジャンプが得意、おたからが少し苦手
- まんべんAI: 全部そこそこ、弱点が小さい

実際に出てくるAIは、この4つの型をもとにランダムな能力差を持ちます。

## 音

効果音とBGMは、外部音源を使わずWeb Audioで生成しています。
スマホの制約に合わせて、最初のタップ後に鳴り始めます。

## 配置

サーバーにこのフォルダをそのままアップロードすれば動きます。
チラシのQRコードは、アップロード先の`index.html`またはフォルダURLに向けてください。
